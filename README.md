>Woodpecker是华东师范大学数据科学与工程学院（简称“DaSE”）研发的一款数据库测试框架。
>
<h2> 版本特性</h2>
目前支持的功能和特性：

* 支持分布式数据库Cedar
* 支持MySQL
* 支持SQL功能测试
* 支持性能测试
* 支持Cedar系统测试
* 提供Notepad++语法格式文件（位于tools目录）
